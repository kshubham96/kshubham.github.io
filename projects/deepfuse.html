DeepFuse: Bridging Human-AI Collaboration for Enhanced CNN Models

  • Tech Stack: Data Quality, Annotation, Data Labelling, Python (Data cleaning, Data transformation).
  • ACM Digital Library: Publication

Overview : Presented DeepFuse, an innovative AI solution that bridges the gap between human experts and Convolutional Neural Networks (CNN) for mitigating contextual bias. DeepFuse enables efficient model enhancement through local explanations, offering the following capabilities:

  1. Efficient browsing of local explanations to identify problematic cases.
  2. Adjustment of attention boundaries for problematic cases.
  3. Fine-tuning the model using attention adjustments.
  4. Evaluation of model changes.

Responsibilities :

  • Curated and cleansed a 5,000-image dataset from the Microsoft COCO dataset for gender classification.
  • Led data annotation and labeling efforts as the 'Data Curator.'
  • Developed Python code to implement CNN, Region-based CNN (R-CNN), and Mask R-CNN models for image classification.
  • Collaborated in a 6-member team for research, including literature review, dataset selection, UI design, and methodology development.

Achievements :

  • Curated and ensured data quality for 5,000+ research images.
  • Applied R-CNN and base object segmentation models to achieve optimized accuracies of 83.2% and 89.2%.
  • Conducted beta-testing and user testing with 10 data scientists for a computer vision tool.
